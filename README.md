# PYluxOS
this is a python made sandbox (fake) linux
which is 100% open source and free for everyone

Its features:
* the 'py' package manager - it uses the packages found in the /packages folder on this github repo
* regular updates - it often gets regular updates and bugfixes
* active package updates - there are active updates for available 
* sandbox mode - dosent exit your fake PYlux distro
* resource saving - using this is much more resourcesaving than running an real vm
* listening to community - i listen to community requests or ideas

Dependencies:
* Python 3.13+
* Windows 10-11 / Linux
* pip (optional)
